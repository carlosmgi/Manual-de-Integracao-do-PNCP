Retificar Parcialmente Ata de Registro de Preço
===============================================

Serviço que permite retificar parcialmente os dados de uma ata de Registro de Preço. Atenção: Não será permitida a retificação do atributo "possibilidadeAdesao" de “true” para “false” da Ata quando a mesma possuir órgãos não participantes e/ou contratos de não participantes vinculados
