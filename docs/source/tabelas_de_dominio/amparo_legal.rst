Amparo Legal 
============

Para consulta aos amparos legais acessar a página no Portal PNCP pelo endereço `https://pncp.gov.br/app/entidades-dominio <https://pncp.gov.br/app/entidades-dominio>`_. Para ajudar na integração com o PNCP 
foram disponibilizados os serviços para consulta de amparo/os legal/is a seguir.
